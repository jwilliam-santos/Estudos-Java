# Estudos-Java

Este repositório foi criado para registrar toda a minha evolução, códigos, anotações e desafios práticos durante o meu aprendizado da linguagem **Java**. 

O objetivo principal aqui é dominar a sintaxe fortemente tipada, entender a fundo a Programação Orientada a Objetos (POO) e construir uma base sólida de engenharia de software.

---

## 📚 Fontes de Estudo & Metodologia

Para aprender de verdade e não cair na armadilha de apenas copiar código, estou combinando teoria conceitual, mentalidade de arquitetura e muita repetição prática através de três pilares:

* **🧠 Livro: Use a Cabeça! Java (3ª Edição)** Utilizado para entender os conceitos de forma visual e profunda. O foco aqui é aprender a "pensar em Java", entender como a JVM (Java Virtual Machine) funciona, como a memória gerencia os objetos e como estruturar classes de forma inteligente.
* **🏋️ Prática: w3resource (Java Exercises)** A fôrma de gelo do aprendizado. Utilizo a enorme lista de exercícios do *w3resource* (Basic, Data Types, Conditional Statements) para massacrar a sintaxe na base da repetição, fixar o uso de ponto e vírgula `;`, chaves `{}` e treinar a tipagem estrita.
* **💻 Meus Projetos & Laboratório** Scripts de testes, desafios resolvidos e, eventualmente, pequenos jogos ou sistemas em modo texto criados para aplicar os conceitos aprendidos no livro.
